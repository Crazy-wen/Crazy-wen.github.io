中文字体环境问题

在用TexShop编译排版数学建模论文模版时，出现以下错误

Package fontspec Error: The font SimSun cannot be found

意思是缺少SimSun字体

解决

从网上下载这些字体，然后在Mac上安装(打开即后点击安装)
http://fontzone.net/ 这是字体下载网站

再次尝试编译
再次出现字体错误

Package fontspec Error: The font simKai cannot be found

但是Mac中是带有楷体的。那到底是什么原因呢？原来，楷体在 Mac 中的名字是 KaiTi 而不是 SIMKAI （这一点可以从字体册中看到）。因此我们，需要在 ctex 的设置中修改一下字体名。需要修改的文件在/usr/local/texlive/2019/texmf-dist/tex/latex/ctex/fontset/ctex-fontset-windowsnew.def

在终端中输入以下命令,修改此文件需要sudo权限

sudo vi /usr/local/texlive/2019/texmf-dist/tex/latex/ctex/fontset/ctex-fontset-windowsnew.def

将simkai.ttf全部换成KaiTi
并将.cls文件的simkai.ttf换成KaiTi

.cls文件与.sty文件

.cls和.sty文件是用来提高LaTeX的排版效果的补充文件。它们分别用命令\documentclass{…}和命令\usepackage{…}来装载到LaTeX文件。.cls文件通常被称为“classes”（也就是“类”），而.sty文件被称为“style files”（样式文件）,或者就简单地称其为“packages”（包）。
